# 📋 ProposalFlow

> Dari Deal ke Duit dalam Satu Alur.

All-in-one client workflow: proposal → contract → invoice → payment.

## Fitur

- **Proposal Builder** — Buat & kirim proposal ke klien
- **Contract Workflow** — Auto-generate contract setelah proposal di-approve
- **Auto-Invoice** — Contract signed → invoice otomatis + payment instructions
- **Payment Tracking** — Record & track pembayaran masuk
- **Dashboard** — Timeline view semua proyek aktif

## Tech Stack

- Node.js + Express
- PostgreSQL
- Dark UI dashboard

## Quick Start

```bash
cp .env.example .env
npm install
npm run db:migrate
npm run dev
```
